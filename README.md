# Petbook

![](/src/assets/app-icon.svg)

Repositório Privado do Petbook.

## Como Rodar.

Como o projeto foi criado com create-react-app, basta fazer:
```js
yarn install // Pra instalar as dependências
```
E logo depois:
```js
yarn start
```

Simples e fácil.
